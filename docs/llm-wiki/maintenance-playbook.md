# Maintenance Playbook

1. 更新 raw 留档。
2. 更新 wiki 概念页。
3. 运行 `spark-harness sync`。
4. 运行 `spark-harness check`。
5. 稳定经验再回写 skill。
6. 不要把单一项目的测试文档提交进 harness 仓库。
